Introduction

Clarity Calendar is a desktop application developed using Python and PyQt5. It combines a calendar with an event scheduler and a to-do list manager, providing a unified platform for organizing your schedule and tasks.

Key Features

  • Calendar View: Visual calendar to navigate through dates.
  • Event Management: Add, edit, and delete events with details like title, date, time, description, priority, and reminders.
  • To-Do List Management: Add, edit, and delete tasks with optional deadlines and priorities.
  • Reminders: Set reminders for events to receive desktop notifications.
  • Data Persistence: Automatically saves your events and tasks locally.
  • Import/Export: Import and export your data in JSON format.
  • Keyboard Shortcuts: Quick access to features using keyboard shortcuts.

Installation

System Requirements

  • Operating System: macOS (10.13 High Sierra or later recommended)
  • Processor: Intel-based Mac or Apple Silicon (with Rosetta 2)
  • Python and Dependencies: Not required for the bundled application
  • Disk Space: Approximately 50MB free space. The app size is around 28MB.

Downloading Clarity Calendar

Clarity Calendar is available as a bundled macOS application (.app) compressed into a ZIP file.

  1. Download the ZIP File

    • Access the download link provided for Clarity Calendar.
    • Click on the link to download the ClarityCalendar.zip file to your Mac.

Installing on macOS

Since Clarity Calendar is not code-signed or notarized, you may encounter security warnings when attempting to run the application. Follow the steps below to install and run the app safely.

Step 1: Unzip the Downloaded File

  • Locate the downloaded ClarityCalendar.zip file, typically in your Downloads folder.
  • Double-click the ZIP file to extract the ClarityCalendar.app application bundle.

Step 2: Move the Application to the Applications Folder (Optional)

  • For easy access, drag and drop the ClarityCalendar.app into your Applications folder.

Step 3: Opening the Application

When you try to open the app for the first time, macOS may prevent it from opening because it is from an unidentified developer. To open the app:

  1. Use Control-Click to Open

    • Navigate to the location of ClarityCalendar.app (Applications folder or where you extracted it).
    • Control-click (or right-click) on ClarityCalendar.app.
    • Select "Open" from the context menu.

    • Confirm Opening the App

    • A warning dialog will appear stating that the app is from an unidentified developer.

    • Click "Open" to confirm you want to run the application.

Alternative Method: Adjust Security Settings

If Control-clicking does not work, you can adjust your security settings:

  1. Open System Preferences

    • Click on the Apple menu () in the top-left corner.
    • Select "System Preferences".

    • Go to Security & Privacy

    • Click on "Security & Privacy".

    • Allow Apps from Unidentified Developers

    • In the "General" tab, you may see a message indicating that ClarityCalendar.app was blocked.

    • Click the "Open Anyway" button next to this message.
    • You may be prompted again to confirm. Click "Open".

Note: Adjusting these settings allows you to open Clarity Calendar but maintains your Mac's security for other applications.

Important Security Information

  • Unidentified Developer Warning: This warning appears because the app is not code-signed or notarized by Apple.
  • Safe to Open: If you trust the source from which you downloaded Clarity Calendar, it is safe to open the app.
  • Future Openings: After following these steps, you won't need to repeat them. You can open Clarity Calendar like any other app.

Getting Started

Launching the Application

After installing, you can launch Clarity Calendar by:

  • From Applications Folder: Double-click ClarityCalendar.app in your Applications folder.
  • From Spotlight Search:
    • Press Command (⌘) + Spacebar to open Spotlight.
    • Type "Clarity Calendar" and press Enter.

Main Interface Overview

The interface is divided into two primary sections:

  1. Left Panel: Contains the calendar and toolbar.
  2. Right Panel: Contains tabs for Events and To-Do List.

Left Panel

  • Toolbar: Buttons to add new events and tasks.
  • Calendar Widget: Interactive calendar to navigate dates and view events.

Right Panel

  • Events Tab: Lists all events.
  • To-Do List Tab: Lists all tasks.

Calendar Features

Viewing the Calendar

  • Navigation: Use the calendar to select dates.
  • Highlighted Dates: Dates with scheduled events are highlighted in light blue.
  • Current Date: The current date is marked.
  • Viewing Events on a Date: Click on a date to view events scheduled for that day. A dialog will display the events.

Adding Events

Method 1: Using the "Add Event" Button

  1. Click the "Add Event" button in the toolbar.
  2. The Add Event dialog appears.

Method 2: Right-Click on the Calendar

  1. Go to and then right-click on a date in the calendar.
  2. Select "Add Event to Selected Date" from the context menu.
  3. The Add Event dialog appears with the date preselected.

Add Event Dialog Fields

  • Title: Enter the event title. (Required)
  • Date: Select the event date.
  • Time: Use the time picker to set the event time.
  • Description: Provide additional details about the event.
  • Priority: Choose from High, Medium, or Low.
  • Recurrence: Select how often the event recurs:
    • None
    • Daily
    • Weekly
    • Monthly
  • Reminders: Add multiple reminders.
    • Click "Add Reminder".

    • Enter the reminder time in **HH

      format (e.g., 09:30). - Reminders will appear in the list. - Deleting Reminders: Right-click on a reminder and select "Delete Reminder"**.

The Reminder functionality is not yet fully implemented, but we should have that running in the next version!

Saving the Event

  • Click "Save" to add the event.
  • A confirmation message will appear in the status bar.

Editing and Deleting Events

Editing an Event

  1. Navigate to the Events tab.
  2. Right-click on the event you wish to edit.
  3. Select "Edit Event" from the context menu.
  4. Make the desired changes in the dialog.
  5. Click "Save" to apply changes.

Deleting an Event

  1. Navigate to the Events tab.
  2. Right-click on the event you wish to delete.
  3. Select "Delete Event" from the context menu.
  4. Confirm deletion when prompted.

Event Reminders

  • Setting Reminders: Add reminders in the Add/Edit Event dialog.
  • Receiving Notifications: At the specified reminder times, a desktop notification will appear with the event title.
  • Note: The application must be running to receive reminders.

The Reminder functionality is not yet fully implemented, but we should have that running in the next version!

To-Do List Features

Adding Tasks

  1. Click the "Add Task" button in the toolbar.
  2. The Add Task dialog appears.

Add Task Dialog Fields

  • Title: Enter the task title. (Required)
  • Set Deadline: Optionally set a deadline.
    • Check the "Set Deadline" box.
    • Select the deadline date.
  • Priority: Choose from High, Medium, or Low.
  • Completed: Check if the task is already completed.

Saving the Task

  • Click "Save" to add the task.
  • A confirmation message will appear in the status bar.

Editing and Deleting Tasks

Editing a Task

  1. Navigate to the To-Do List tab.
  2. Right-click on the task you wish to edit.
  3. Select "Edit Task" from the context menu.
  4. Make the desired changes in the dialog.
  5. Click "Save" to apply changes.

Deleting a Task

  1. Navigate to the To-Do List tab.
  2. Right-click on the task you wish to delete.
  3. Select "Delete Task" from the context menu.
  4. Confirm deletion when prompted.

Event and Task Management

Context Menus

  • Events:
    • Right-click on an event in the Events tab to access options:
      • Edit Event
      • Delete Event
  • Tasks:
    • Right-click on a task in the To-Do List tab to access options:
      • Edit Task
      • Delete Task
      • Mark as Completed

Searching and Filtering

  • Events:
    • Use the search bar at the top of the Events tab.
    • Type keywords to filter events.
  • Tasks:
    • Use the search bar at the top of the To-Do List tab.
    • Type keywords to filter tasks.

Importing and Exporting Data

Exporting Data

  1. Go to the File menu in the menu bar.
  2. Select "Export Data".
  3. Choose a location to save the JSON file.
  4. Click "Save".
  5. A confirmation message will appear upon successful export.

Importing Data

  1. Go to the File menu in the menu bar.
  2. Select "Import Data".
  3. Browse to the JSON file containing your data.
  4. Click "Open".
  5. Confirm data import when prompted.
  6. The application will refresh with the imported data.

Keyboard Shortcuts

Clarity Calendar supports several keyboard shortcuts to enhance productivity:

  • Add Event
    • Ctrl + E
  • Add Task
    • Ctrl + T
  • Reset Calendar View to Today's Date
    • Ctrl + G
  • Delete Selected Item
    • Ctrl + D

Tips and Best Practices

  • Data Persistence: Your events and tasks are saved automatically. No need to manually save your data.
  • Set Multiple Reminders: Use the reminder feature to set multiple notifications for important events.
  • Prioritize: Assign priorities to your events and tasks to help manage your time effectively.
  • Mark Tasks as Completed: Keep track of your progress by marking tasks as completed.
  • Export Regularly: Keep backups of your data by exporting it periodically.
  • Stay Organized: Regularly review and update your events and tasks to keep your schedule current.

Known Limitations

  • Recurring Events Logic: While you can select recurrence for events, the application currently does not automatically generate future occurrences. Recurring event functionality is limited.
  • Reminders for Tasks: Reminders can only be set for events, not tasks.
  • Visual Indication on Calendar: Clicking on a date shows events in a dialog, but events are not displayed directly on the calendar.
  • No Undo Option: Deletion of events and tasks is permanent once confirmed.
  • Time Zone Awareness: The application does not handle time zones explicitly. All times are assumed to be in your local time zone.
  • Limited Exception Handling: Errors may occur if unexpected inputs are provided.
