Introduction

Clarity Clips is a clipboard manager that keeps a history of your clipboard content, allowing you to access, search, and manage previous clipboard entries with ease. Whether you're a developer, writer, or anyone who frequently copies and pastes content, Clarity Clips helps you streamline your workflow by providing quick access to your clipboard history.

Why is this different from a simple 'copy' and 'paste' function with the Cmd+C and Cmd+V? Well, let us say you are a lawyer with 10 tabs open, each with a different case. You want to select text from a case and put it into a master file. And the same case may have different passages that you need. Or you are a researcher with 10 PDF articles open, and you want to make notes easily. If you just use the copy and paste functionality, you must copy each item and then paste that item, then copy the next item and paste that one, and so on. But with Clarity Clips, just keep doing your copy actions, without worrying about the pastes - you won't lose out the first copy just because you made the second copy action... After you have done all your 100 (well, up to 500 with Clarity Clips!) copy actions, you will see all the copied content safely stored in Clarity Clips. And then you can tag each entry, search for content within your copied entries, export all your copies, copy back each entry, basically do what you want to...

Getting Started

System Requirements

  • Operating System: macOS 10.13 High Sierra or later
  • Processor: Intel-based Mac
  • Disk Space: At least 50 MB of free space
  • Internet Connection: Required for downloading the application

Download and Installation

Clarity Clips is distributed as a macOS application bundle compressed in a ZIP file. Since the application is not code-signed or notarized, additional steps are required to install and run it.

Important: macOS security features may prevent the app from running because it is not code-signed or notarized. Please follow the instructions carefully to install and launch the application.

Steps to Download and Install:

  1. Download the ZIP File:

    • Obtain the Clarity Clips ZIP file from the official download link provided by the developer.

    • Locate the Downloaded ZIP File:

    • By default, it should be in your Downloads folder.

    • Extract the ZIP File:

    • Double-click the ZIP file to extract it.

    • You should now see the Clarity Clips.app application bundle.

    • Move the Application to the Applications Folder:

    • Drag and drop Clarity Clips.app into your Applications folder for easy access.

    • Alternatively, you can run it from the extracted location, but storing it in the Applications folder is recommended.

First Launch

Since Clarity Clips is not code-signed or notarized, macOS will prevent it from opening by default due to security settings. You need to manually allow the application to run.

Steps to Open Clarity Clips for the First Time:

  1. Attempt to Open the Application:

    • Go to your Applications folder.
    • Double-click Clarity Clips.app.

    • macOS Security Warning:

    • You will see a message saying:

      "Clarity Clips.app" cannot be opened because the developer cannot be verified.

    • Click "Cancel".

  2. Open via System Preferences:

    • Open System Preferences by clicking the Apple menu  and selecting "System Preferences...".

    • Click on "Security & Privacy".

    • In the "General" tab, you will see a message:

      "Clarity Clips.app" was blocked from use because it is not from an identified developer.

    • Click the "Open Anyway" button.

  3. Confirm Opening the Application:

    • A new prompt will appear:

      macOS cannot verify the developer of "Clarity Clips.app". Are you sure you want to open it?

    • Click "Open".

  4. Grant Necessary Permissions:

    • Clarity Clips requires access to monitor your clipboard.
    • If prompted, grant the application the necessary permissions.
    • You may need to enter your administrator password to allow changes.

    • Application Launches:

    • Clarity Clips should now open and appear in your Dock.

    • The main window will display, and the application is ready to use.

Note: You only need to perform these steps the first time you open Clarity Clips. Subsequent launches will not require this process.

Alternative Method Using Control-Click:

If you prefer, you can use the following method to open the application without navigating through System Preferences.

  1. Control-Click the Application:

    • Go to your Applications folder.
    • Hold down the Control key and click on Clarity Clips.app.
    • Select "Open" from the context menu.

    • Confirm Opening the Application:

    • A prompt will appear similar to the one above.

    • Click "Open" to proceed.

Main Features

Clipboard Monitoring

Clarity Clips continuously monitors your system clipboard for any changes. Whenever you copy new content, the application automatically records it in the clipboard history unless it's a duplicate of the most recent entry. For best use, make sure Clarity Clips runs at login - either configure your system to run it at login, or simply open the app (and then you can minimise the app window without any worry): the app does its work in the background...

Supported Content Types:

  • Plain text
  • HTML content (extracts plain text)
  • File paths to:
    • PDF documents
    • Word documents (.docx)
    • Excel files (.xlsx) The app will store the plain text from all these formats.

Clipboard History Table

The main window displays a table containing your clipboard history with the following columns:

  1. Timestamp: The date and time when the content was copied.
  2. Content Preview: A truncated preview of the clipboard content (up to 100 characters).
  3. Tags: Any tags associated with the entry.

Table Features:

  • Row Selection: Click on a row to select it for actions like copying or deleting.
  • Sorting: Click on the column headers to sort the entries.
  • Resizing: Columns automatically resize to fit content.

Search Functionality

At the top of the main window, there's a search bar allowing you to filter entries based on keywords or tags.

  • How to Use:
    • Enter any keyword or tag in the search bar.
    • The table updates in real-time to show only matching entries.
    • Search is case-insensitive and looks for matches in all columns.

Copying Entries

You can copy the full content of any clipboard entry back to your clipboard.

  • Steps:
    1. Select the desired entry in the table.
    2. Click the "Copy Selected" button.
      • Shortcut: Press Command+C (⌘+C).
    3. A confirmation message appears indicating the content has been copied.

Note: The application temporarily ignores clipboard changes initiated by itself to prevent duplication.

Tagging Entries

Add tags to your clipboard entries for better organization and easier searching.

  • Steps:
    1. Select the desired entry in the table.
    2. Click the "Add Tag" button.
      • Shortcut: Press Command+T (⌘+T).
      • Alternative: Double-click on the entry in the table.
    3. An input dialog appears.
    4. Enter one or more tags, separated by commas.
    5. Click "OK" to save the tags.
    6. The tags are added to the entry and displayed in the Tags column.

Deleting Entries

Remove unwanted entries from your clipboard history.

  • Steps:
    1. Select the entry you wish to delete.
    2. Click the "Delete Selected" button.
      • Shortcut: Press Command+D (⌘+D).
    3. A confirmation dialog appears.
    4. Click "Yes" to confirm deletion.
    5. The entry is removed from the history.

Exporting History

You can export your entire clipboard history to a text file for backup or archival purposes.

  • Steps:
    1. Click the "Export History" button.
      • Shortcut: Press Command+E (⌘+E).
    2. A save file dialog appears.
    3. Choose the location and filename for the export.
    4. Click "Save".
    5. A confirmation message appears upon successful export.

Clearing History

Clear all entries from your clipboard history.

  • Steps:
    1. Click the "Clear History" button.
    2. A confirmation dialog appears.
    3. Click "Yes" to confirm.
    4. All entries are removed, and the table is cleared.

Note: This action is irreversible unless you have exported your history.

System Tray Integration

Clarity Clips minimizes to the system tray, allowing it to run in the background without occupying space on your Dock.

  • Tray Icon Features:

    • Tooltip: Hover over the tray icon to see "Clarity Clips."
    • Context Menu: Right-click the tray icon to access options:
      • Restore: Brings the main window back into view.
      • Clear History: Clears the clipboard history.
      • Exit: Closes the application.

    • Tray Icon Interaction:

    • Single Click: Restores the application window.

    • Notifications: Displays messages when the application is minimized.

Note: When you close the main window, the application minimizes to the tray instead of exiting.

Keyboard Shortcuts

Enhance your productivity by using the following keyboard shortcuts:

  • Copy Selected Entry: Command+C (⌘+C)
  • Add Tag to Entry: Command+T (⌘+T)
  • Delete Selected Entry: Command+D (⌘+D)
  • Export History: Command+E (⌘+E)
  • Refresh History Table: Command+R (⌘+R)

Usage Tips:

  • Ensure the main window is focused to use the shortcuts.
  • Shortcuts can be used in combination with mouse actions for efficiency.
  • On non-Mac keyboards, the Command key is often represented as the ⌘ symbol.

Settings and Configuration

Maximum Entries

Clarity Clips maintains up to 500 entries in your clipboard history.

  • Warning Threshold: At 450 entries, a warning message appears suggesting you export your history.
  • Automatic Trimming: Once the limit is reached, older entries are deleted to make room for new ones.

File Storage

  • History File Location:

    ~/Library/Application Support/ClarityClips/clipboard_history_with_tags.txt

  • Log File Location:

    ~/Library/Logs/ClarityClips/clipboard_manager.log

Note: These directories are created automatically if they do not exist.

Handling Different Content Types

Clarity Clips intelligently processes different types of content:

  • Plain Text: Stored as-is.
  • HTML Content:
    • Extracts and stores the plain text.
  • File Paths:
    • Recognizes file paths copied to the clipboard.
    • Supports text extraction from:
      • PDF Files: Uses PyPDF2 to extract text.
      • Word Documents (.docx): Uses python-docx to extract text.
      • Excel Files (.xlsx): Uses openpyxl to extract text.

Important:

  • If text extraction fails, the file path itself is stored.
  • The application does not process images or other non-text content.

Notifications and Warnings

  • Clipboard Change Detection:

    • Logs clipboard changes.
    • Displays minimal notifications to avoid disruptions.

    • History Limit Warning:

    • When nearing the maximum entry limit, a warning prompts you to export your history.

    • This warning is displayed only once per session.

Logging and Error Handling

  • Logging:

    • All significant actions and errors are logged to help with troubleshooting.
    • Logs include timestamps and error details.

    • Error Handling:

    • The application gracefully handles exceptions.

    • User-friendly error messages are displayed when issues occur (e.g., failing to read or write the history file).

Conclusion

Clarity Clips aims to simplify your clipboard management by providing a straightforward and efficient tool to track and manage your clipboard history. By familiarizing yourself with its features and shortcuts, you can significantly enhance your productivity.

Frequently Asked Questions (FAQs)

Q1: Does Clarity Clips record sensitive information like passwords?

A: Clarity Clips records all clipboard content, including sensitive information. Be cautious when copying confidential data. You can delete specific entries or clear the entire history if needed.

Q2: How can I prevent the application from closing when I click the close button?

A: By default, closing the main window minimizes Clarity Clips to the system tray. To fully exit the application, right-click the tray icon and select "Exit".

Q3: Can I customize the maximum number of entries stored?

A: In the current version, the maximum number of entries is fixed at 500. Future updates may include customizable settings.

Q4: Why am I seeing duplicate entries in my history?

A: Clarity Clips checks for duplicates against the most recent entry only. If you copy the same content after copying different content in between, it will be recorded again.

Q5: The application isn't extracting text from some PDFs. What can I do?

A: Text extraction depends on the PDF's content and structure. Some PDFs may not allow text extraction or may contain scanned images of text. In such cases, the file path is stored instead.

Q6: I'm not receiving any notifications. Is this normal?

A: Clarity Clips minimizes notifications to avoid distractions. It shows critical alerts like the history limit warning or confirmation messages for actions like copying or deleting entries.

Q7: How do I update Clarity Clips to the latest version?

A: Since Clarity Clips is distributed as a standalone application, you need to download the latest version from the official source and replace your existing application.

Q8: I received a warning about the application not being verified. Is it safe to proceed?

A: Yes, Clarity Clips is safe to use. The warning appears because the application is not code-signed or notarized. Follow the instructions in the First Launch section to open the application.

Q9: Can I run Clarity Clips on Windows or Linux?

A: The current version is designed for macOS due to directory paths and system tray implementation. Modifications are required to run it on other operating systems.
