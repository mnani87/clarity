Keyboard shortcuts are provided to enhance your productivity.

Getting Started

Installing Clarity Editor on macOS

Clarity Notes is available as a downloadable .app bundle compressed in a .zip file. Follow the steps below to install and launch the application on your Mac. Please note that since the app is not code-signed or notarized, you may receive security warnings when attempting to open it.

Steps to Install and Launch Clarity Editor:

  1. Download the Application:

  2. Obtain the ClarityEditor.zip file from the provided download link.

  3. Extract the Application:

  4. Locate the downloaded ClarityEditor.zip file, typically found in your Downloads folder.

  5. Double-click the .zip file to extract it. This will create the ClarityEditor.app bundle.

  6. Move the Application to Applications Folder (Optional):

  7. For easier access, you may move ClarityEditor.app to your Applications folder: Drag and drop the ClarityEditor.app into the Applications folder in Finder.

  8. Launching Clarity Editor for the First Time:

  9. Since the app is not code-signed or notarized, macOS may prevent it from opening directly. Follow these steps to open it: a. Using Finder:

    • Open Finder and navigate to the location of ClarityEditor.app (either in Downloads or Applications).
    • Control-click (or right-click) on ClarityEditor.app.
    • Select Open from the context menu.
    • A warning dialog will appear stating that the app is from an unidentified developer.
    • Click Open to confirm that you want to open the application. b. Adjusting Security Settings (Alternative Method):
    • If the above method does not work, you can adjust your security settings: Open System Preferences and click on Security & Privacy. In the General tab, you may see a message about ClarityNotes.app being blocked. Click the Open Anyway button next to this message. Confirm by clicking Open in the subsequent dialog.

  16. Subsequent Launches:

  17. After the initial launch, you can open Clarity Editor normally by double-clicking the app icon.

Important Notes:

  • Security Considerations: Because the app is not code-signed or notarized, macOS cannot verify its integrity. Only proceed if you trust the source of the application. Always exercise caution when running applications from unidentified developers.
  • System Requirements: macOS Version: Clarity Editor requires macOS 10.13 High Sierra or later. Python and Dependencies: All necessary components are bundled within the app; you do not need to install Python or additional packages.

User Interface Overview

Clarity Editor features a clean and intuitive interface divided into the following components:

  • Main Text Editor: The central area where you type and edit your document.
  • Format Toolbar: Located at the top, contains text formatting options.
  • Additional Toolbar: Below the Format Toolbar, provides paragraph styling, table management, and file operations.
  • Status Bar: At the bottom, displays messages about actions performed.

Text Formatting

Bold

  • Purpose: Make selected text bold.
  • How to Use: Select the text you want to bold. Click the Bold button on the Format Toolbar. Shortcut: Press Ctrl + B (or Command + B on macOS) to toggle bold formatting.

Italic

  • Purpose: Italicize selected text.
  • How to Use: Select the text you want to italicize. Click the Italic button on the Format Toolbar. Shortcut: Press Ctrl + I (or Command + I on macOS) to toggle italic formatting.

Underline

  • Purpose: Underline selected text.
  • How to Use: Select the text you want to underline. Click the Underline button on the Format Toolbar. Shortcut: Press Ctrl + U (or Command + U on macOS) to toggle underline formatting.

Left Align

  • Purpose: Align text to the left margin.
  • How to Use: Place the cursor in the paragraph or select multiple paragraphs. Click the Left button on the Format Toolbar.

Center Align

  • Purpose: Center text horizontally.
  • How to Use: Place the cursor in the paragraph or select multiple paragraphs. Click the Center button on the Format Toolbar.

Right Align

  • Purpose: Align text to the right margin.
  • How to Use: Place the cursor in the paragraph or select multiple paragraphs. Click the Right button on the Format Toolbar.

Justify

  • Purpose: Distribute text evenly between the left and right margins.
  • How to Use: Place the cursor in the paragraph or select multiple paragraphs. Click the Justify button on the Format Toolbar.

Font and Size Selection

Changing Font

  • Purpose: Change the typeface of your text.
  • Available Fonts: Avenir, Arial, Book Antiqua, Charter, Franklin Gothic, Garamond, Gill Sans, Helvetica, Optima, Palatino, Times New Roman.
  • How to Use: Select the text you wish to change. Choose a font from the Font Selector dropdown on the Format Toolbar.

Adjusting Font Size

  • Purpose: Change the size of your text.
  • How to Use: Select the text you wish to resize. Use the Font Size Selector (spinner) on the Format Toolbar to choose a size between 8 and 48 points.

Line Spacing

  • Purpose: Adjust the spacing between lines within a paragraph.
  • Options: 0.8, 0.9, 1, 1.15, 1.25, 1.3, 1.5, 2 (Default is 1.15).
  • How to Use: Place the cursor in the paragraph or select the text you want to adjust. Choose a value from the Line Spacing Selector on the Additional Toolbar.

Paragraph Spacing

  • Purpose: Adjust the spacing before and after paragraphs.
  • How to Use: Place the cursor in the paragraph or select the text you want to adjust. Use the Paragraph Before Selector and Paragraph After Selector (both are spinners with values from 0 to 50 points) on the Additional Toolbar. Before: Sets the space above the paragraph. After: Sets the space below the paragraph.

Lists and Tables

Bulleted Lists

  • Purpose: Create a list with bullet points.
  • How to Use: Place the cursor where you want to start the list. Click the Bullets button on the Format Toolbar. Type your list items, pressing Enter after each item.

Numbered Lists

  • Purpose: Create a list with numbered items.
  • How to Use: Place the cursor where you want to start the list. Click the Numbering button on the Format Toolbar. Type your list items, pressing Enter after each item.

Inserting a Table

  • Purpose: Add a table to organize data.
  • How to Use: Click the Insert Table button on the Additional Toolbar. A dialog will appear asking for the number of rows and columns. Enter the desired number of rows and columns (between 1 and 100). Click OK to insert the table at the cursor's location.

Modifying a Table

  • Purpose: Add or remove rows and columns in an existing table.
  • How to Use: Place the cursor inside the table you want to modify. Click the Modify Table button on the Additional Toolbar. A dialog will appear: Enter the number of rows to add (positive number) or remove (negative number). Enter the number of columns to add or remove in the same way. Click OK to apply the changes. Note: You cannot remove more rows or columns than the table currently has.

File Operations

New Document

  • Purpose: Create a new, blank document.
  • How to Use: Click the New button on the Additional Toolbar or press Ctrl + N (or Command + N on macOS). If you have unsaved changes, you will be prompted to save before proceeding.

Open File

  • Purpose: Open an existing document.
  • Supported Formats: HTML (.html), Text (.txt), ODF Text (.odt).
  • How to Use: Click the Open button on the Additional Toolbar or press Ctrl + O (or Command + O on macOS). Select the file you wish to open from the file dialog. If you have unsaved changes, you will be prompted to save before opening a new file.

Save File

  • Purpose: Save the current document.
  • How to Use: Click the Save button on the Additional Toolbar or press Ctrl + S (or Command + S on macOS). If it's a new document, you will be prompted to choose a file name and format (see Save As). If the document has been saved before, it will overwrite the existing file.

Save As

  • Purpose: Save the current document under a new name or format.
  • Supported Formats: HTML (.html), Markdown (.md), Text (.txt), PDF (.pdf), ODF Text (.odt).
  • How to Use: Click the Save As button on the Additional Toolbar or press Ctrl + Shift + S (or Command + Shift + S on macOS). A dialog will prompt you to select the desired file format. Choose the format and click OK. In the file dialog, specify the file name and location. Click Save to save the document.

Keyboard Shortcuts

Enhance your efficiency with the following shortcuts:

  • Bold: Ctrl + B (Windows) / Command + B (macOS)
  • Italic: Ctrl + I / Command + I
  • Underline: Ctrl + U / Command + U
  • New Document: Ctrl + N / Command + N
  • Open File: Ctrl + O / Command + O
  • Save File: Ctrl + S / Command + S
  • Save As: Ctrl + Shift + S / Command + Shift + S

Tips and Tricks

  • Selecting Text: Double-click to select a word, triple-click to select a paragraph.
  • Undo/Redo: While not explicitly provided, you can use standard shortcuts (Ctrl + Z / Command + Z for Undo and Ctrl + Y / Command + Y for Redo) if supported by your system.
  • Adjusting Multiple Paragraphs: To apply formatting to multiple paragraphs, select all the text you wish to format before using the toolbar options.
  • Navigating Tables: Use the Tab key to move to the next cell and Shift + Tab to move to the previous cell.
  • Focus on Editor: If toolbar actions seem unresponsive, click back into the editor area to ensure it is focused.

Troubleshooting

Security Warning on Launch

  • Issue: macOS displays a warning that ClarityEditor.app cannot be opened because it is from an unidentified developer.
  • Solution: Follow the steps in Launching Clarity Editor for the First Time to open the app safely.

Unsaved Changes Prompt

  • Issue: You're prompted to save unsaved changes when creating a new document or opening a file.
  • Solution: Choose Yes to save your changes, No to discard them, or Cancel to return to the document.

File Not Opening

  • Issue: Unable to open a file.
  • Possible Causes: Unsupported file format. File is corrupted.
  • Solution: Ensure the file is in one of the supported formats and is not corrupted.

Formatting Not Applied

  • Issue: Text formatting doesn't seem to apply.
  • Solution: Ensure you've selected the text or placed the cursor appropriately before applying formatting.

Table Modification Errors

  • Issue: Unable to remove rows or columns from a table.
  • Possible Cause: Attempting to remove more rows or columns than exist.
  • Solution: Only remove up to the number of existing rows or columns.
